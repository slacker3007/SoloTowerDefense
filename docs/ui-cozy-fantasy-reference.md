# Cozy Fantasy UI Reference

## Design Tokens

- Primary background: `#221a1f`
- Panel surface: `#2f2630`
- Panel border/accent: `#bda67a`
- Primary button: `#4f3f38`
- Button hover: `#6a5648`
- Primary text: `#f8efe0`
- Muted text: `#d9c8ac`
- Success text: `#b7e3a1`
- Error text: `#f28b82`

## Core Components

- Cozy card panel with warm border and high legibility text.
- Large action buttons with clear verbs.
- Tabs and section labels for discoverability.
- Feedback line for validation and conflict states.
- Dimmed overlays for pause and run-end context.

## UX Copy Patterns

- Use direct verbs: `Resume`, `Settings`, `Retry`, `Back to Menu`.
- Show constraints clearly: `This key is already assigned.`
- Show progress context: `Wave`, `Enemies`, `Role`.
